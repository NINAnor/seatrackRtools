# Get paths to all startup Excel files

This function retrieves paths to all Excel files in the "Starttime files
and stored loggers" subdirectory of the sea track folder.

## Usage

``` r
get_startup_paths()
```

## Value

A character vector containing paths to all Excel files in the specified
subdirectory.

## Examples

``` r
if (FALSE) { # \dontrun{
all_xlsx_files <- get_startup_paths()
} # }
```
