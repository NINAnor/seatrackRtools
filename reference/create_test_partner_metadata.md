Create test partner metadata Generates a test partner metadata object
with predefined encounter data, restart times, and return times. This
function is useful for testing functions that require a partner metadata
object without needing to read from an actual file.

## Usage

``` r
create_test_partner_metadata()
```

## Value

A list representing a test partner metadata object, containing encounter
data, restart times, and return times.
