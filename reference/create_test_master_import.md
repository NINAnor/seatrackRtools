# Create test master import workbook Generates a test workbook with metadata and startup sheets for testing master import functionality.

Create test master import workbook Generates a test workbook with
metadata and startup sheets for testing master import functionality.

## Usage

``` r
create_test_master_import(file_path = NULL)
```

## Arguments

- file_path:

  Optional file path to save the generated workbook. If NULL, the
  workbook is not saved to disk.

## Value

A list containing the workbook object and the data frames for metadata
and startup sheets.
