# Prepare GLS position data for database upload

Function to prepare GLS position data from a specified directory for
upload to the Sea Track database.

## Usage

``` r
gls_prepare_folder_upload(gls_directory_path)
```

## Arguments

- gls_directory_path:

  Path to the directory containing GLS position data files.

## Value

A dataframe containing the prepared GLS position data ready for database
upload.
