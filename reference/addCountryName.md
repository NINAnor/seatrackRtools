# Add country name to colony locations based on nearest NaturalEarth feature

Add country name to colony locations based on nearest NaturalEarth
feature

## Usage

``` r
addCountryName(coldf)
```

## Arguments

- coldf:

  Data frame with colony locations (must contain columns 'col_lon' and
  'col_lat')

## Value

Data frame with added 'sovereignty' and 'country' columns
