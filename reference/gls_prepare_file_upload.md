# Prepare a single GLS position data file for database upload

Function to prepare a single GLS position data file for upload to the
Sea Track database.

## Usage

``` r
gls_prepare_file_upload(gls_file_path)
```

## Arguments

- gls_file_path:

  Path to the GLS position data file.

## Value

A dataframe containing the prepared GLS position data ready for database
upload.
