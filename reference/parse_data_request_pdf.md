Parse a single PDF data request file and extract relevant information
This function reads a PDF file, extracts specific information such as
the applicant's name, project title, study area, species involved, and
other details, and compiles this information into a structured format.

## Usage

``` r
parse_data_request_pdf(pdf_path)
```

## Arguments

- pdf_path:

  The file path to the PDF data request file.

## Value

A data frame containing the extracted information from the PDF file.
